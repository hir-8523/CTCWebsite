Roll Marking Machine

Roll Marking Machine

Ideal for marking cylindrical (Solid or hollow) components of Steel, Aluminium, Brass, etc. using CHANDRIKA Interchangeable Steel Types (IST) and Logo Marking Dies. The Steel Types or Stamp Marks are mounted in Holder fixed to the machine carriage. Perfect for marking specific components such as hose end fittings, bushes, drills, taps, engine valves, spark plugs, cutting tools, diamond tools, nozzles, sanitary fittings, gas lighters, and studs.

Low set-up time. Easy operation. A simple hand lever operates the carriage producing a uniform, legible marking. For Pneumatic Model, the Cylinder operates the carriage. Marking depth is precisely set by adjusting the vertical dove-tail slide.

Machine Holder for Roll Marking Machine

When the mark you need to stamp will frequently change, a Steel Type Holder is ideal. Our Machine Type Holders are also compatible with Steel Type characters of various sizes. Using a Machine Type Holder increases efficiency and supports accuracy and uniform imprints of identical depth. You can use it to arrange individual characters for spelling out different names, codes, or serial numbers, etc.

Features

  • Designed for accurate IST Type location and alignment
  • Positive locking of IST Types
  • Enables quick change of IST Types & high interchangeability
  • Long-lasting
